【仮想バナー】

静かに過ごしたいソロキャンパー向けのキャンプ場の特集in静岡

  • TOOL : PhotoShop

  • DATE : 1日（5時間）

  • 目的

    ・静かにゆっくり過ごしたいキャンパーに静かに過ごせるキャンプ場があることを知ってもらいたい＆来てほしい
    ・静岡県にキャンプをしてきてほしい、静岡県の良さを知ってほしい

Point

Point1 ”しずかな”をポイントに文字の字体を細めにし、文字の間隔を広めにとり空間の静かさを表現。

Point2 静岡県の富士山をイメージしたシンプルなテーマイメージを作成しました。文字が見えづらい箇所は白いグラデーションを重ね、濃い色の文字を引き立てるようにしました。
