OverView

【仮想ウェブサイトを想定】

東京都内にオープンした、フレンチレストランのWebサイト
本場で培った技術と日替わりのメニューが味わえるフレンチレストランです。

  • TOOL : Googleドライブ(概要まとめ)/PhotoShop/XD/VScode

  • DATE : 約4週間
    （企画まとめ3日/フレームワーク1週間/デザインワーク１週間/コーディング１週間）

  • ターゲット

    30代〜40代後半

  • 目的

    ・新規客、予約数を増やす
    ・認知数を増やす
    ・店舗の場所が分かりにくいためWebで表示させたい

  • 課題

    知る人ぞ知るといったような場所にお店があるので、認知している方を増やし集客につなげたい。

  • その他

    ・メニューが日によって変わる
    ・オーナーはフランスで10年修行をし、日本でお店を構えて3年になる。
    ・近所のお店に比べ値段は少し安め

Color

  • #FDF9F5

  • #65645C

  • #445549

  • #8E4330

トーンを抑え全体的に落ち着いた印象に。
メニューを表記する際に黒板を使用するため、深い色の緑と、ワインを連想させた赤を使用。
お店の雰囲気から少し柔らかい、親やすい印象にしたかったため背景色は白っぽくなりすぎないように設定。

Point

Point1 おひとり様や少人数のお客さまをターゲットとしているため、トップイメージに店内の画像を使用し、親やすさを表現しています。

Point2 個人経営のため、定休日以外の休みやイベント出店など期間限定の食事などあるため見落としがないようトップ画像と被るように設定。

Point3 メニューについては個別（instagram）で管理、更新予定のため確認用のURLを設定。みていただくよう促すとともに気に入った方にフォローをしていただくとともに、認知していただくようにする。
